# MCP‑Sim ✨  

Composable scientific micro‑models with one JSON‑RPC request.

MCP‑Sim turns any validated equation, ML weight file, or empirical dataset model into a plug‑and‑play web service. Chain them, validate them, and pay only for compute.

* Try the public sandbox → [sandbox.mcp‑sim.com](https://sandbox.mcp-sim.com)
* → **[Quick Start](quick-start)** in 10 minutes

## Key Features

* 🧩 **Micro-models:** one task each (climate, hydro, crop…)
* 🔌 **Model Control Protocol (MCP):** standardized JSON-RPC interface
* 💸 **Credit ledger:** pay per call, transparent pricing
* 👥 **Community marketplace:** discover and compose models

## Platform Architecture

```
Front-end (React) ←→ Router (Flask) ←→ Model Nodes (Any Language)
     ↓                    ↓                      ↓
• Visualizations      • Auto Discovery      • Custom Logic
• Model Browser       • Load Balancing      • Health Checks  
• Dynamic Forms       • Peer Validation     • JSON-RPC API
```
